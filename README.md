# GrimLandsKeeper
Open Source Dungeon Keeper 2 Reimplementation

### Status
It is barely started

### Tools and technologies used
* C++11
* OpenGL 3.2
* Box2D
* cJSON
* Dear ImGui
* STBI
* GLM math library
* GLFW
* GLEW
